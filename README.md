# blinkitclone
# 🛒 Blinkit Clone App

This application is a Blinkit-inspired clone built as a personal project to practice and enhance Android development skills.
It is not an official product and is not associated with Blinkit in any way.

---

## 📌 About the Project

This project is built to practice Android development and understand how a real-world grocery delivery app works.

---

## 🚀 Features

* 🏠 Home Section (Product Listing)
* 🔍 Search Functionality
* 🛒 Cart System (Add/Remove items)
* 👤 Profile Section
